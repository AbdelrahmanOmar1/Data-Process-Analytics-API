- 💰 Profit Analysis

## 🧠 How It Works

1. Upload CSV data (sales / purchases)
2. Backend processes and stores analytics
3. Prisma ORM aggregates insights
4. Frontend displays:
   - Lists (Top Sales, Dead Stock, Profit)
   - Charts (dynamic visualization)
5. Search & filtering applied in real-time
